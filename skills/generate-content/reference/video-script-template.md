# Nexudus Unlocked — Video Script Template

Video scripts are written for the **Nexudus Unlocked** YouTube series. These are external-facing,
customer/operator-audience walkthrough videos. Every script follows the same structure.

Real examples are in `examples/video-scripts/`. Load one before generating — it takes precedence
over this template.

---

## Series conventions (apply to every script)

- **Series name:** Nexudus Unlocked
- **Audience:** Space operators and customers (external)
- **Tone:** Conversational but clear. Written for the ear. Contractions fine.
- **Delivery:** Voiceover narration over screen recording
- **Screen cues:** Every scene change gets a `[square bracket annotation]` describing what should
  be on screen. Reference actual GIF/MP4 filenames if known; use descriptive placeholders if not.
- **Title card slides:** Used as section dividers — `[White screen: Section heading text]`
- **Orange slides:** Used for the series intro and major feature title cards

## Screen cue formats

Use these consistently throughout:

| Cue | When to use |
|---|---|
| `[Show [path navigation] and navigate to [Path]]` | When demonstrating navigation |
| `[Show [GIF filename or description]]` | For recorded feature demos — use actual filenames if known |
| `[White screen: [heading text]]` | Section title card / divider |
| `[Orange Slide - [text]]` | Series intro or major section title |
| `[Show screenshot - [description]]` | For static screenshots |
| `[INTRO - Unlocked clip]` | Opening series branding clip — always first |
| `[Outro]` | Closing series branding — always before the standard close line |

## Narration rules

- **Write for the ear** — read every sentence aloud. If it trips you up, rewrite it.
- **One action per beat** — don't say "Click X and then navigate to Y" — split into two sentences.
- **Short sentences** — aim for ≤20 words. Complex ideas get their own sentence.
- **Contractions are natural** — "Let's", "you'll", "we'll", "it's" all fine.
- **Present tense** — "Click Save" not "You'll need to click Save".
- **Second person** — "you can", "you'll see", "select your".
- **No marketing language** — "highly requested & powerful new feature" → describe what it does.
- **Navigation paths** — still bold: **Settings > Companion Apps > NexBoard**
- **UI elements** — still bold when used as actions: "Click **Save changes**"

---

## Tone — before/after

| ❌ Avoid | ✅ Use instead |
|---|---|
| "Today, we are going to take a look at a highly requested & powerful new feature" | "Today, we're looking at [Feature] — [what it does in one sentence]." |
| "Let's jump right into it" | "Let's start with [first thing]." |
| "As you can see, what we've done here is..." | "The system [does X]." |
| "You may ask 'How is this different from...'" | "Unlike [X], [Feature] [does Y instead]." |
| "That's about it for this episode!" | Use the standard close line verbatim — see below. |

## Standard close line (use verbatim)

> That's about it for this Nexudus Unlocked! As always, if you have any questions, please reach
> out to our Support team or check out our Knowledge Base, help.nexudus.com.

Do not paraphrase or vary this line.
